Getting Started

Installation

git clone https://github.com/botshelomokoka/anya-core.git
cd anya-core
cargo build --release

Basic Configuration

Configure your environment by creating a .env file:

NETWORK_TYPE=testnet
LOG_LEVEL=info
ENABLE_AI=true

API Reference

Core API

  • GET /api/v1/health - Health check endpoint
  • GET /api/v1/wallet/status - Wallet status
  • POST /api/v1/transaction - Create transaction

AI Endpoints

  • POST /api/v1/ai/analyze - Analyze transaction patterns
  • GET /api/v1/ai/insights - Get AI insights
